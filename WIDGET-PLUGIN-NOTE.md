# Widget Plugin Notu — Tarımİklim Hava Widget’ı (HalFiyatları)

HalFiyatları gibi yüzeyler için Tarımİklim hava widget’ı embed edilebilir.

## Yeni ortak paket

- Paket: `@agro/ecosystem-weather-widget`
- Kaynak: `packages/ecosystem-weather-widget`

## Kullanım (ileride)

- Brand: `haldefiyat`
- `apiBase`: `https://tarimiklim.com/api/v1`

İframe embed devam edebilir; native component widget istenirse bu paket referans alınacak.
